#Pokedex

Pequeno projeto onde se aplica a arquitetura MVVMC, consumo de API, favoritagem entre outros. Showcase the capacidades

##Funcionalidades
- Listagem de Pokemons
- Detalhes de Pokemons
- Favoritos Locais com CoreData
- Paginação Infinita

##Tecnologia
- Swift 5
- UIKit
- CoreData
- PokeAPI
- JSON Parsing
